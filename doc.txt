1- Create virtual-host in apache2 with DocumentRoot kmc/public
2- change the domain name in .htaccess to the given virtual-host
3- Create Database and defined it in config/config.php file
4- create app_user table with the following schema:
  'UserId'
  'Username'
  'Firstname'
  'Lastname'
  'Password'
  'Email'
  'SubscriptionDate'
  'LastUpdate'
  'LastLogin'
  'GroupId'
  'Status'

5- create new user in app_user
6- call the url virtual-host/app-admin
